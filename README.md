ÖZET  
Bu projede otelcilik sektörüne yönelik responsive bir web sitesi tasarlanmış ve “Proje Otel” 
markası oluşturulmuştur. Site; ana sayfa, hakkımızda, odalarımız, duyurular, birimler, 
misyon–vizyon, iletişim, rezervasyon ve istatistik sayfalarını içermektedir. Ayrıca admin 
paneli tasarlanmıştır. Proje HTML5, CSS3, JavaScript, Bootstrap, Chart.js ve modern web 
teknolojileriyle geliştirilmiştir. 

I. GİRİŞ  
Bu proje, Web Tasarımı dersi kapsamında öğrencilerin hazır bir şablonu özelleştirerek 
kurumsal yapıda profesyonel bir web sitesi geliştirmesini amaçlamaktadır. Otelcilik sektörü 
seçilmiş ve Proje Otel markası tasarlanmıştır. Proje çok sayfalı yapı, kurumsal kimlik, mobil 
uyumluluk ve kullanıcı deneyimi odaklı olacak şekilde geliştirilmiştir. 

II.  PROJE İSTERLERİ  
 Responsive tasarım  
 Menü, slider, duyurular  
 Oda listesi + detay sayfaları  
 Hakkımızda, Misyon–Vizyon, Birimler sayfaları  
 Google Maps + iletişim formu  
 Chart.js grafik sayfası  
 Admin duyuru ekleme paneli  
 Ortak header & footer yapısı  

III.  ŞABLON SEÇİMİ VE ÖZELLEŞTİRME  
Proje, “Hotelier HTML Template” üzerine kurulmuştur. Tema otel sektörüne göre 
özelleştirilmiş; tipografi, logo, menü yapısı ve görseller güncellenmiştir. Sayfa düzenleri 
modern otel sitelerine uygun şekilde yeniden tasarlanmıştır. 

IV.  SİTE SAYFALARI  
 Ana Sayfa: Slider, duyurular, yönlendirme butonları içerir. 
 Odalar: Tüm odalar, Suit, Standart, Aile odası listesi sunulmuştur. 
 Rezervasyon: Tarih seçimi, kişi sayısı, kişisel bilgiler bölümlerini içerir. 
 Duyurular: Modal detay gösterimi mevcuttur. 
 Hakkımızda  
 Misyon–Vizyon  
 Birimler: Restoran, Spa, Etkinlik hizmetlerini içerir. 
 İletişim: Google Maps + form bulunmaktadır. 
 Sayılarla Otelimiz: Chart.js grafiklerini içerir. 

V. ADMIN PANEL  
Admin panel arayüz seviyesinde tasarlanmıştır. Başlık, açıklama, tarih ve dosya ekleme 
bölümleri içerir ve CMS mantığını simüle eder. 

VI.  SORUNLAR VE ÇÖZÜMLER  
 Menü hizalama sorunları → Bootstrap düzenlemesi  
 Görsel orantısızlıkları → object-fit: cover  
 Grafik responsive sorunları → chart-container düzenlemeleri  
 Footer hizalama → Flexbox + min-height  

VII.  SONUÇ  
Bu projede otelcilik sektörüne uygun kurumsal ve profesyonel bir web sitesi geliştirilmiştir. 
Proje isterleri tamamen karşılanmış olup kullanıcı deneyimi, mobil uyumluluk ve çok sayfalı 
site yapısı başarıyla uygulanmıştır.
